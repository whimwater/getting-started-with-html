The World Wide Web was invented in the early 90's and is made up of HTML. HTML stands for 'Hyper Text Mark-up Language'. Programming works by connecting a computer and chosen browser with the Internet and Servers. This is done thru HTTP (hyper text transfer protocols).

HTML consists of text content, mark up, and references to other docs (e.g. images, videos, etc.)

HTML Tags
*Please note: I've added the the text form of tags within single quotes intentially to show work.

HTML mark-up is made up of tags. Tags have an opening tag '(<em>)' and a closing tag '(</em>)'. '<b>' is the BOLD tag. '<em>' is the ITALICS tag.
If you don't close a tag, it will continue throughout the text until you close it.

HTML Attributes
An anchor '<a>' is an HTML attribute and is used to make links.
Using '<a href="www.link.com"> display text </a>', you create a link within text. Example: Google.

Another HTML attribute is used to add an image within a line of text. Using '<img src="link to image">' adds an image inline with text.

White Space
I have been adding 'white space' in different ways throughout these notes. I can add white space by adding a line break tag '<br>' or by adding a paragraph tag '<p>'. Both create white space, but are considered different types of elements. The Paragraph tag '<p>' is a block element and creates an imaginary box around content, the paragraph 'block' can be assigned a height and width. "Inline elements" like line breaks '<br>' stay within the line of text, as we saw with the image element.

HTML Documents
An HTML document is the entirety of the code for a given document. An HTML Document requires a 'doctype', 'head', and a 'body'. The 'head'contains meta-data, css, etc., and the title of the document. The content is contained within the 'body'. All of the content in this document has been added to the "body" of my HTML code.
